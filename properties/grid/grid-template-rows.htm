grid-template-rows

  • 版本：CSS3
  • 媒体：视觉

语法：

grid-template-rows：none | <track-list> | <auto-track-list>

默认值：none

适用于：网格容器

继承性：无

动画性：是，如果列表长度匹配

计算值：指定值

取值：

none：
不明确行。隐式网格行将自动生成。
<track-list>：
定义显式网格行的列表。可以使用长度值、百分比、fr单位、minmax()函数、repeat()函数等。
<auto-track-list>：
定义包含自动重复行的列表。

说明：

设置或检索网格容器的行定义。
  • 示例：定义三行网格，高度分别为 100px, auto, 1fr

    Code：

    .grid { display: grid; grid-template-rows: 100px auto 1fr; }
  • 对应的脚本特性为gridTemplateRows。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
