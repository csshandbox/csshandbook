grid

  • 版本：CSS3
  • 媒体：视觉

语法：

grid：<' grid-template '> | <' grid-template-rows '> / [ auto-flow && dense? ] <' grid-auto-columns '>? | [ auto-flow && dense? ] <' grid-auto-rows '>? / <' grid-template-columns '>

默认值：看各分拆属性

适用于：网格容器

继承性：无

动画性：看各分拆属性

计算值：看各分拆属性

说明：

复合属性。设置或检索网格容器的显式网格属性以及隐式网格属性。
  • 示例：

    Code：

    .grid { display: grid; grid: 100px 300px / 3fr 1fr; }
  • 对应的脚本特性为grid。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
