网格布局 Grid Layout

属性 版本 继承性 描述
grid CSS3 无 复合属性。设置或检索网格容器的显式网格属性以及隐式网格属性。
grid-template CSS3 无 复合属性。设置或检索网格容器的列、行和区域。
grid-template-rows CSS3 无 设置或检索网格容器的行定义。
grid-template-columns CSS3 无 设置或检索网格容器的列定义。
grid-template-areas CSS3 无 设置或检索网格容器的区域定义。
grid-gap CSS3 无 复合属性。设置或检索网格容器的行和列的间隙。
grid-row-gap CSS3 无 设置或检索网格容器的行间隙。
grid-column-gap CSS3 无 设置或检索网格容器的列间隙。
grid-auto-rows CSS3 无 设置或检索隐式网格行的尺寸。
grid-auto-columns CSS3 无 设置或检索隐式网格列的尺寸。
grid-auto-flow CSS3 无 设置或检索自动布局算法的工作方式。
grid-area CSS3 无 复合属性。设置或检索网格项的区域位置。
grid-row CSS3 无 复合属性。设置或检索网格项的行位置。
grid-row-start CSS3 无 设置或检索网格项的行起始位置。
grid-row-end CSS3 无 设置或检索网格项的行结束位置。
grid-column CSS3 无 复合属性。设置或检索网格项的列位置。
grid-column-start CSS3 无 设置或检索网格项的列起始位置。
grid-column-end CSS3 无 设置或检索网格项的列结束位置。

紫色 链接表示该属性在CSS3有修改或者增加了新的属性值；橙色 链接表示该属性是CSS3新增属性
