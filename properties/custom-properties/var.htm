var()

  • 版本：CSS3
  • 媒体：视觉

语法：

var()：var( <custom-property-name> [, <declaration-value> ]? )

适用于：所有元素

继承性：无

取值：

<custom-property-name>：
自定义属性名，以 -- 开头。
<declaration-value>：
后备值（fallback value），当自定义属性无效或未定义时使用。

说明：

使用自定义属性（变量）。
  • 示例：定义并使用变量

    Code：

    :root { --main-color: #06c; } .box { color: var(--main-color); }

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
